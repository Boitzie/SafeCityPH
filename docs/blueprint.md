# **App Name**: Makati Guardian

## Core Features:

- Real-time Incident Monitoring: Display a live, auto-updating summary of all reported incidents in Makati City with filters.
- Report Verification: Allow administrators to review and validate new incident reports.
- Detailed Report View: Provide a comprehensive view of each incident report, including location, images, reporter information, timeline, and notes.
- Admin Notes: Enable administrators to add, edit, delete, and view notes related to each incident. These notes will sync in real time.
- Incident Timeline Generation: Generate the first draft of an incident timeline using the incident report details as a tool. This would save administrators time.
- Profile Settings: Allow administrators to manage their profile information, change their password, and upload a profile photo.
- Department Assignment: Enable reassignment of responsibility to appropriate departments.

## Style Guidelines:

- Primary color: Deep blue (#003049) for a professional and trustworthy feel.
- Background color: Light gray (#F0F4F7), almost white, to ensure visual clarity and a clean aesthetic. Its hue is closely matched to that of the primary color.
- Accent color: Teal (#26A69A) for interactive elements and highlights, providing a contrasting yet harmonious touch. This color is an analogous of the primary color in hue, but also offers strong differentiation from the primary in terms of brightness and saturation.
- Body and headline font: 'Inter' (sans-serif) for a clean and modern user interface.
- Use clear and easily recognizable icons to represent different incident categories and actions.
- Maintain a clean and modern layout with consistent spacing and padding (16-24px) to ensure a visually appealing and user-friendly interface.
- Incorporate subtle animations for transitions and feedback to enhance the user experience without being distracting.
